# Kafka Confluent Cloud e Banco de Dados PostgreSQL

Este projeto demonstra como usar o Kafka Confluent Cloud para produzir e consumir mensagens em um tópico Kafka, e como integrar essas mensagens com um banco de dados PostgreSQL usando Python.

## Pré-requisitos

Antes de executar este projeto, certifique-se de ter o seguinte configurado:

- Uma conta na plataforma Confluent Cloud com um cluster Kafka configurado.
- Um banco de dados PostgreSQL em execução, com as credenciais e a URL de conexão corretas.

Para ver este projeto em ação, confira o vídeo tutorial no YouTube: Integrando Kafka Confluent Cloud com PostgreSQL em Python.

https://www.youtube.com/watch?v=qwT8jEvAAms
